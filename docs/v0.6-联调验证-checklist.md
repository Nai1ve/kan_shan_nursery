## 10. 故障排查

| 症状 | 原因 | 处置 |
| --- | --- | --- |
| 后端启动时报 `kanshan_shared` 未找到 | shared-python 未在 sys.path | 确认 `conftest.py` 在仓库根；或在服务 venv 内 `pip install -e packages/shared-python` |
| `/api/v1/profile/me` 503 `SERVICE_NOT_READY` | profile-service 没起来或 gateway 配置错 | `curl http://127.0.0.1:8010/health` + 检查 `GATEWAY_READY_SERVICES` 环境变量 |
| 前端 gateway 模式下白屏 | fetch CORS 失败或 envelope 解包出错 | 打开浏览器 Console；若是 CORS，确认 gateway 运行在 127.0.0.1 而非 localhost；若是解包，检查 `output/logs/api-gateway-*.jsonl` 看是否返回 error envelope |
| live 模式下所有接口 401 | 凭证没填 / 写错 | `ZhihuAuthError` 会在日志 `zhihu_call_failed` 事件里显示；核对 `services/config.yaml` |
| live 模式下 429 | 命中真实知乎限流或本地 quota 耗尽 | 等一下再试；或在 config.yaml 提升 quota（本地 quota 是保护伞，真实额度仍受知乎后端限制） |
| OAuth callback 报 `OAuth app_id/app_key missing` | OAuth 凭证没填 | 见 §9.5，先填 app_id/app_key/redirect_uri |
